# 🏀 Score-Board

A clean, interactive sports scoreboard application built with **HTML**, **CSS**, and **JavaScript**. This project allows users to track points and fouls for two teams (Home and Guest) in real-time.
> As a beginner, still learning 🚀 I built this project to practice my fundamental JavaScript skills, like DOM manipulation and state management.

## 🚀 Features
- **Team Scoring:** Incremental buttons for +1, +2, and +3 points.
- **Foul Tracking:** Dedicated counters for team fouls with safeguards to prevent negative numbers.
- **Game Reset:** A "Restart" button to instantly zero out all scores and fouls for a new match.

## 🛠️ Built With
- **HTML5:** Semantic structure for the scoreboard.
- **CSS3:** Custom styling for a professional sports-broadcast look.
- **JavaScript (ES6):** Basic logic for score updates and state management.
